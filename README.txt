Registration Form Project
Overview
This project is a simple registration form that collects user information, including personal details, account type, and additional information. The form is designed to be user-friendly and easy to navigate, with clear labels and instructions.

Features
Collects user's first name, last name, email, and password
Allows users to select their account type (personal or business)
Enables users to upload a profile picture
Collects user's age (in years)
Asks how the user heard about the service (with options for freeCodeCamp News, YouTube Channel, Forum, or Other)
Provides a text area for users to write a bio
Requires users to accept the terms and conditions before submitting the form
Technical Details
The form is built using HTML5 and CSS3
The form uses the https://register-demo.freecodecamp.org endpoint for submission
The password field has a pattern requirement of at least 8 characters, with only lowercase letters and numbers allowed
The age field has a minimum value of 13 and a maximum value of 120
The referrer field uses a select element with predefined options
The bio field uses a textarea element with a placeholder text
Getting Started
To use this registration form, simply open the index.html file in a web browser. Fill out the form with the required information and click the "Submit" button to submit the form.
